Mass Spectrometry Toolbox (MST) is a collection of tools to quantify a specific analyte using mass spectrometry data.

Mass spectrometry instruments generate a large amount of data. A typical 3D mass spectrum is shown bellow using the m/z parameter collected over time (min). The intensity of each peak is represented on the OZ axis. This 3D representation is the first level. Each peak from this level can have it's own 3D mass spectrum linked to it if it's intensity is above a specified threshold. The maximum depth (i.e., number of levels) can go up to nine levels deep. The data is highly structured within each level. However, the presence of extra deeper level(s) at specific items results in an increased data complexity.

The raw data file obtained from the mass spectrometer is first converted to the an mzXML file.
The mzXML file is then used to quantify the analyte. The quantification is based on a standard curve (also called a callibration curve). The standard curve is created using a relevant compound similar to the analyte under study. A typical standard for an analyte could be the deuterium labeled analyte or just a very similar molecule.

Each user has its own workspace and the flow of the data analysis was designed to be simple and intuitive.

STEP 1 - Create a new project


A project is created when the user initiates a new analysis. A project contains all the setting files, test file, and the report files.
Users can create new projects by selecting the button on the main menu.

STEP 2 - Create a setting for the project

Each project needs a setting. A setting is just a collection of files (we will call them setting files). All files from a setting share the mass to charge ratio (m/z) of the standard analyte under question. A new setting can be created and subsequently linked to the active project by selecting the button on the main menu. Each file represents an analysis (similar to the test file linked to the project) of the analyte under question at different concentrations.


STEP 3 - Add files to the newly created setting

A typicall setting includes a collection of 7 or more files. Each file is linked to a specific concentration. Each concentration is a added as a dillution of the stock concentration of the analyte. Values range from 1 (stock molar concentration) down to 10-9. As a consequence, each setting file needs to have an analyte concetration added at the time of the upload.

The m/z is set for all files when the setting is created since is identical for all files within the setting. After all files that are associated to one setting are uploaded and the corresponding anlyte concentrations are saved, the standard curve can be generated with a click of a button.

After the standard curve is generated, the test file can be uploaded and analyzed. After the analysis, a link to the comprehensive report will be available in the main project space.
